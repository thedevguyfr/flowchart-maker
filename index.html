Quick Help

Keyboard Shortcuts

  • Ctrl/Cmd + Z - Undo
  • Ctrl/Cmd + Y - Redo
  • Ctrl/Cmd + D - Duplicate selected
  • Delete/Backspace - Delete selected
  • Shift + Click - Multi-select
  • Double Click - Edit node text
  • Ctrl/Cmd + A - Select all

Actions

  • Drag nodes from toolbar to canvas
  • Click toolbar item to add at center
  • Drag nodes to move them
  • Drag node corners to resize
  • Click and drag from connection handles to create connectors
  • Right-click and drag to pan canvas
  • Zoom with mouse wheel or pinch
  • Right-click nodes for context menu
